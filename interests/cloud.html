Cloud Computing

What is Cloud Computing

Cloud Computing is an Information Technology that enables users to access pools of information, and services through the internet. Cloud computing provides a simple way to access servers, storage, databases and a broad set of application services over the Internet. It is used by both bussinesses and people as their everyday storage service. Cloud services are becoming increasingly popular as they are a easy, secure and effective way to store data, as the data can be accessed anywhere with a internet connection.In order to avail of these services most of cloud businessses will provide you with a cloud on a pay as you go pricing.

Types of cloud services available

Based on a cloud location, we can classify cloud as:public,private,hybrid and a community cloud. These clouds all work in diiferet ways and are useful in different ciricumstances.

  • Public cloud This mean's that the whole computing infrastructure is located on the premises of a cloud computing company. The location remains, thus, separate from the customer and has no physical control over the infrastructure. As public clouds use shared resources, they do excel mostly in performance, but are also most vulnerable to various attacks.
  • Private cloud This means using a cloud infrastructure (network) solely by one customer/organization. It is not shared with others The companies have an option of choosing an on-premise private cloud as well, which is more expensive, but they do have a physical control over the infrastructure. The security and control level is highest while using a private network.
  • Hybrid cloud This means using both private and public clouds, depending on their purpose. For example, public cloud can be used to interact with customers, while keeping their data secured through a private cloud
  • Community cloud implies an infrastructure that is shared between organizations, usually with the shared data and data management concerns. For example, a community cloud can belong to a government of a single country. Community clouds can be located both on and off the premises.
